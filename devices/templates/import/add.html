CSV Upload Guide

Follow these steps to upload your CSV file:

  1. Create an Excel file with the following headings:
    • centre_code
    • department
    • hardware
    • system_model
    • processor
    • ram_gb
    • hdd_gb
    • serial_number
    • assignee_first_name
    • assignee_last_name
    • assignee_email_address
    • device_condition
    • status
  2. Ensure you enter the centre codes assigned to you, not the centre names.
  3. Ensure the department column contains department codes (e.g., IT_DEPT), not department names.
  4. Convert the Excel file to CSV format.
  5. Upload the CSV file using the form above.

Note: Only CSV files are accepted for upload.
